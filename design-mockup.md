# Momo's Pretzels - Design Mockup & Wireframe Documentation

## Brand Overview
**Business Name:** Momo's Pretzels
**Tagline:** "Baked with Love"
**Brand Personality:** Warm, rustic, homemade, artisanal, family-feel
**Primary Audience:** Farmers market shoppers looking up vendors on mobile devices

---

## 1. Color Palette

### Primary Colors
```
Golden Crust       #D4A574  (Main brand color - warm pretzel gold)
Deep Brown         #6B4423  (Text, headers - rich chocolate brown)
Warm Cream         #F5E6D3  (Backgrounds, cards)
Butter Yellow      #F4D58D  (Accents, highlights)
```

### Secondary Colors
```
Salt White         #FFFEF9  (Primary background, clean space)
Sesame Brown       #8B6F47  (Secondary text, borders)
Caramel            #C68642  (Hover states, CTAs)
Sage Green         #9CAF88  (Fresh, natural accent for farmers market feel)
```

### Semantic Colors
```
Success Green      #7FA86A  (Form success states)
Error Red          #C55A47  (Form validation errors)
Info Blue          #7B9FB0  (Information banners)
```

### Usage Guidelines
- **Backgrounds:** Salt White (#FFFEF9) for main, Warm Cream (#F5E6D3) for sections
- **Text:** Deep Brown (#6B4423) for headers, Sesame Brown (#8B6F47) for body
- **CTAs:** Golden Crust (#D4A574) with Caramel (#C68642) hover
- **Accents:** Butter Yellow (#F4D58D) for highlights, Sage Green (#9CAF88) for freshness cues

## 2. Typography System

### Font Pairing (Google Fonts)

**Primary Font - Headers:**
```
Font Family: 'Playfair Display'
Weights: 600 (Semi-Bold), 700 (Bold)
Purpose: Headlines, page titles, logo text
Rationale: Classic, warm serif that conveys artisanal quality
```

**Secondary Font - Body:**
```
Font Family: 'Nunito'
Weights: 400 (Regular), 600 (Semi-Bold), 700 (Bold)
Purpose: Body text, navigation, buttons
Rationale: Friendly, rounded sans-serif that's highly readable on mobile
```

**Accent Font - Special Elements:**
```
Font Family: 'Satisfy'
Weight: 400 (Regular)
Purpose: Tagline "Baked with Love", handwritten notes
Rationale: Handwritten feel for personal, homemade touch
```

## 6. Mobile-First Considerations

### Touch Targets
- Minimum size: 44px x 44px (Apple HIG guideline)
- Spacing between interactive elements: 8px minimum
- Primary CTA buttons: 48px height minimum

### Performance
- Lazy load images below the fold
- Optimize hero image (WebP format, max 1200px width)
- Critical CSS inline for above-fold content
- Load Google Fonts async

### Mobile Navigation
- Hamburger menu with smooth slide-in (300ms)
- Close button clearly visible (top-right)
- Menu items large enough for thumbs (56px height)
- Current page highlighted

### Forms on Mobile
- Large input fields (48px height)
- Labels inside fields with floating effect
- Virtual keyboard considerations (email/tel input types)
- Submit button sticky at bottom on small screens
- Clear error messages above/below fields

### Images
- Responsive images with srcset
- Mobile: 375px, Tablet: 768px, Desktop: 1440px
- Aspect ratios maintained across breakpoints
- Pretzel photos: 4:3 ratio
- Hero images: 16:9 ratio

### Typography on Mobile
- Minimum body text: 16px (prevents zoom on iOS)
- Line-height: 1.5 for body text (readability)
- Max line length: 65 characters (desktop)
- Adequate spacing between sections (40px+)

### Gestures
- Swipeable review carousel
- Pull-to-refresh on market schedule
- Pinch-to-zoom on pretzel photos
- Tap to expand FAQ items

## 7. Accessibility Requirements

### Color Contrast
All text meets WCAG AA standards:
- Deep Brown (#6B4423) on Salt White (#FFFEF9): 8.9:1
- Sesame Brown (#8B6F47) on Salt White: 5.2:1
- Salt White on Deep Brown: 8.9:1
- Golden Crust on Deep Brown: 4.8:1

### Semantic HTML
- Proper heading hierarchy (H1 ‚Üí H2 ‚Üí H3)
- `<nav>` for navigation
- `<main>` for main content
- `<article>` for pretzel cards
- `<section>` for content sections
- `<footer>` for footer

### ARIA Labels
- `aria-label="Main navigation"` on nav
- `aria-label="Close menu"` on close button
- `aria-current="page"` on active nav item
- `aria-required="true"` on required form fields
- `aria-live="polite"` on form validation messages

### Keyboard Navigation
- Skip to main content link
- Focus visible styles (2px Golden Crust outline)
- Tab order follows visual order
- Escape closes modal/menu
- Enter/Space activates buttons

### Screen Reader Support
- Alt text for all images (descriptive, not "image of")
- Form labels properly associated with inputs
- Error messages announced
- Loading states announced
- Success confirmations announced

### Focus States
```css
:focus-visible {
  outline: 2px solid #D4A574; /* Golden Crust */
  outline-offset: 4px;
  border-radius: 4px;
}
```

## Design Rationale

### Why This Approach Works

**Mobile-First:**
Over 70% of farmers market shoppers use phones to look up vendors on-site. The design prioritizes thumb-friendly navigation, large touch targets, and quick-loading content.

**Warm Color Palette:**
Golden and brown tones evoke freshly baked bread and create appetite appeal. The palette is grounded in the actual product (pretzels) for authenticity.

**Clear Hierarchy:**
Each page has one primary action (Find Us, See Menu, Contact) with clear visual priority through size, color, and positioning.

**Accessibility-First:**
High contrast, semantic HTML, and keyboard navigation ensure the site is usable by everyone, reflecting the inclusive nature of farmers markets.

**Content Simplicity:**
Information-only design keeps focus on what matters: where to find Momo's and what pretzels are available. No e-commerce complexity for a market-based business.

**Trust Signals:**
Reviews, photos of Momo, story section, and authentic imagery build trust and connection before the first market visit.
